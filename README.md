# 🍋 Little Lemon Restaurant Website

A modern and responsive restaurant website built to showcase Mediterranean cuisine, weekly specials, customer experiences, and seamless online reservations.

Designed with a clean UI, vibrant visuals, and mobile-friendly navigation, the project delivers an engaging and smooth browsing experience for users across all devices.

---

## 🌟 Features

- 🍽️ Beautiful landing page for restaurant branding
- 📱 Fully responsive design for mobile, tablet, and desktop
- 🥗 Mediterranean menu showcase
- ⭐ Weekly specials section
- 🖼️ Interactive image gallery
- 💬 Customer testimonials & reviews
- 📅 Online reservation/booking section
- ⚡ Smooth navigation and modern UI/UX

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Responsive Web Design
